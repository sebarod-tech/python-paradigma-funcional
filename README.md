Este programa sirve para procesar las ventas de una empresa.

Analiza las ventas mayores a 1000

Aplica descuentos de acuerdo a la categoría de cada venta

•	Categoría A: 20% descuento 
•	Categoría B: 10% descuento
•	Categoría C: 5% descuento

Calcula los resultados con descuento

Genera información detallada, como cliente, monto original, categoría, monto final.


Este programa transforma datos, aplica reglas y obtiene resultados.

Cómo ejecutar

python actividad2.py
